# Button
Creates a primary, secondary or normal button.

## Examples
<Inline itemSpace="medium">
    <Button primary icon="times" iconLeft="bookmark-r">
        <Text>Primary</Text>
    </Button>
    <Button primary disabled>Disabled</Button>
    <Button secondary>Secondary</Button>
    <Button secondary disabled>Secondary disabled</Button>
    <Button onClick={() => { alert("You clicked me"); }}>Click me</Button>
</Inline>

## Properties

### primary - boolean
Button will be styled as primary button

<Button primary>Primary</Button>

### secondary - boolean
Button will be styled as secondary button

<Button secondary>Secondary</Button>

### disabled - boolean
Disables the button.
<Button primary disabled>Disabled</Button>

### icon - string
Adds an icon next to the button content. Any registered icon can be used.

<Button secondary icon="times">Icon Button</Button>

### iconLeft - string
Adds an icon before the button content. Any registered icon can be used.

<Button secondary iconLeft="times">Left Icon Button</Button>

### as - Component

The button will be rendered with the provided component. Default component is `NailsButton`

```jsx
import CustomButton from './CustomButton'

<Button as={CustomButton}>CustomButton</Button>
```

<Box>
    <Button as={CustomButton}>CustomButton</Button>
</Box>
