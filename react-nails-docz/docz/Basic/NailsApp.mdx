# NailsApp

NailsApp is the main entry point. All other components must be wrapped inside this component. It handles the basic setup and theming. `NailsApp` can be nested to switch themes for parts of the application.

## Example
```
<NailsApp theme={{...}}>
   ...
</NailsApp>
```

## Properties

### theme - object
With `theme` you can customize the basic theme for your needs. Every theme property can be overriden.
