# InputNumber

Creates a number input element 

## Example

<Inline>
    <InputNumberExample changeAmount={0.01} suffix={<Text>€</Text>}/>
</Inline>

## Properties

### changeAmount - number
Input values are restricted to be a fit at least to this amount. All other values get rounded to fit. Default is 1.
