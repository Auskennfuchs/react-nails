# InputPassword

Creates a number input element with up/down arrows as prefix.

## Example

<Inline>
    <InputPassword />
</Inline>

## Properties

### showIcon - string

Sets the icon display when the plain password can be shown to the user.

### hideIcon - string

Sets the icon display when the star password can be shown to the user.
