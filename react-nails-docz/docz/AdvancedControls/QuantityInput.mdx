# InputQuantity

Creates a number input element with up/down arrows as prefix.

## Example

<Inline>
    <QuantityExample changeAmount={0.01} suffix={<Text>€</Text>}/>
</Inline>

## Properties

### increaseIcon - string

Sets the icon diplayed on increase button.

### decreaseIcon - string

Sets the icon diplayed on decrease button.
