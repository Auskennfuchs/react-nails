# Divider

A Divider is a horizontal or vertical line to separate blocks.

## Example

<Column>
    <div>
        Some Text
        <br />
        Some more Text
    </div>
    <Divider />
    <div>New chapter, new text</div>
</Column>

## Properties

### vertical - boolean

If set the line is vertical
